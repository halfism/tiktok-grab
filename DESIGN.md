# DESIGN.md

## 气质与意象
**关键词**：专业、高效、极简、工具感
**场景描述**：一个深夜工作室的界面，屏幕上只有必要的操作区域，暗色背景减少视觉干扰，用户专注于内容采集任务本身。界面像一块干净的玻璃，没有多余的装饰，所有元素服务于核心功能。

## 配色方案
- **主色**：深空灰 (#0F172A) - 背景，营造专业氛围
- **强调色**：品牌蓝 (#3B82F6) - 主要操作按钮、进度指示
- **辅助色**：抖音红 (#FE2C55) / TikTok黑 - 平台标识区分
- **中性色**：浅灰 (#E2E8F0) - 文字、分割线
- **成功色**：翠绿 (#10B981) - 下载成功状态
- **警告色**：琥珀 (#F59E0B) - 处理中状态

## 字体排版
- **字体族**：思源黑体 (Source Han Sans / Noto Sans SC)
- **标题**：16-24px，字重600
- **正文**：14px，字重400
- **说明文字**：12px，字重300
- **代码/链接**：等宽字体 (Monaco / Consolas)

## 页面结构
### 主页面布局
1. **顶部导航栏**：Logo + 平台切换（抖音/TikTok）+ 设置入口
2. **核心操作区**：
   - 链接输入框（支持粘贴、拖拽）
   - 解析按钮（主要CTA）
   - 实时状态反馈
3. **解析结果区**：
   - 媒体预览卡片（视频封面、图集缩略图、音频波形）
   - 元数据展示（标题、作者、时长、分辨率）
   - 操作按钮（下载、复制链接、查看详情）
4. **参数配置面板**（可折叠）：
   - 平台选择
   - 资源类型（视频/图集/音频）
   - 分辨率选项
   - 文件格式
   - 命名规则
5. **批量任务区**（底部或侧边）：
   - 任务列表
   - 进度条
   - 批量操作按钮

## 组件规范
### 链接输入组件
- 圆角：8px
- 边框：2px，默认浅灰，焦点时品牌蓝
- 占位符文字："粘贴抖音或TikTok链接..."
- 支持自动识别链接格式

### 媒体卡片
- 圆角：12px
- 阴影：轻柔投影 (shadow-md)
- 尺寸：视频卡片宽度280px，高度按比例
- 悬停效果：微浮起 + 边框高亮

### 进度指示
- 线性进度条：高度4px，圆角2px
- 状态文字：进度百分比 + 当前任务名
- 颜色变化：等待(灰) → 进行中(蓝) → 成功(绿) → 失败(红)

### 按钮
- 主按钮：品牌蓝背景，白色文字，圆角8px
- 次按钮：浅灰背景，深灰文字
- 禁用状态：降低透明度，cursor: not-allowed

## 交互与状态
- **解析状态**：loading动画 + "正在解析..."文字
- **下载状态**：进度条实时更新 + 速度/剩余时间显示
- **错误处理**：红色提示卡片 + 重试按钮
- **成功反馈**：绿色图标 + "下载完成" + 文件路径展示

## 动效设计
- **页面加载**：元素依次淡入，间隔100ms
- **解析按钮点击**：轻微下沉效果
- **卡片悬停**：transform: translateY(-2px)
- **进度条**：平滑过渡，transition: width 0.3s ease
- **状态切换**：颜色渐变过渡，duration: 200ms

## 设计禁忌
- ❌ 禁止使用过于鲜艳的渐变背景
- ❌ 禁止使用emoji或装饰性图标
- ❌ 禁止卡片内嵌入过多文字信息
- ❌ 禁止使用弹窗打断用户操作流程（使用Toast提示）
- ❌ 禁止在解析过程中隐藏进度反馈
